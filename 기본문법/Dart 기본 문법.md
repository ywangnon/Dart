# 기본 문법



## 주석



```dart
// 이것은 주석. 한 줄 주석

/**
*	이것도 주석. 여러 줄 주석
**/

/// 이것도 주석. 문서 주석
```

- /// 를 메서드나 클래스 정의 위에 작성하면 dartdoc과 같은 문서 생성 ㄷ구를 통해 문서를 자동으로 생성
- dart.dev/tools/dartdoc

void main() {
